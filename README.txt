README - خطوات سريعة لرفع الموقع على GitHub Pages
-------------------------------------------------
1) فك الضغط عن الملف وضع المجلد 'jadwal-react' في مكان تشتغل منه.
2) افتح الطرفية داخل المجلد وشغّل:
   npm install
   npm run build
3) طريقة رفع بسيطة (رفع الملفات كاملة إلى repo):
   - أنشئ Repository جديد على GitHub (اسمه مثلاً: jadwal-mu7adarat)
   - ارفع جميع الملفات داخل المجلد 'jadwal-react' إلى فرع main
   - في GitHub: Settings -> Pages -> Source -> Branch: main -> Folder: / (root) -> Save
   - بعد لحظات سيظهر رابط: https://YOUR_USERNAME.github.io/jadwal-mu7adarat
4) ملاحظة: المشروع مُعد ليعمل محليًا ويدعم التعديل من لوحة الإدارة (password: khaled181)
5) لو تحب أنشره أنا بنفسى إلى gh-pages branch عبر GitHub Actions أو أجهز repo جاهز، أقولك وانا أعمله.
